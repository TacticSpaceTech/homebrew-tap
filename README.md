# Homebrew Tap for Tactic Remote

Control Claude Code from your iPhone/iPad.

## Install

```bash
brew install TacticSpaceTech/tap/tacticremote
```

## Run

```bash
tacticremote
```

Scan the QR code with the [Tactic Remote](https://tacticremote.com) iOS app to connect.
